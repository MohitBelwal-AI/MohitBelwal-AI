; в”Җв”Җв”Җ WHY THIS MATTERS в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
;
;  Python Class:
;  class Motor:
;      power_kw = 7.5       вҶҗ attribute
;      def start(self):     вҶҗ method
;          self.status = "Running"
;
;  PLC Function Block (FB):
;  FUNCTION_BLOCK Motor_FB
;      VAR
;          power_kw : REAL;     вҶҗ attribute
;          status   : BOOL;     вҶҗ attribute
;      END_VAR
;      (* start logic here *)   вҶҗ method (ladder inside FB)
;  END_FUNCTION_BLOCK
;
;  Python Object:
;  pump = Motor()           вҶҗ instantiation
;  pump.start()             вҶҗ method call
;
;  PLC FB Instance:
;  pump : Motor_FB          вҶҗ instantiation
;  pump.EN := TRUE          вҶҗ method call (enable)
;
;  SAME CONCEPT вҖ” OOP exists in PLC too!
;  IEC 61131-3 standard supports full OOP via FB
;
; в”Җв”Җв”Җ FUNCTION BLOCK DEFINITION в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
;
;  FUNCTION_BLOCK Motor_FB
;  (* This is the CLASS definition *)
;
;  VAR_INPUT
;      EN_START    : BOOL;     вҶҗ start command input
;      EN_STOP     : BOOL;     вҶҗ stop command input
;      OL_TRIP     : BOOL;     вҶҗ overload relay
;      STAR_PRESET : INT;      вҶҗ star timer preset (seconds)
;  END_VAR
;
;  VAR_OUTPUT
;      KM_MAIN  : BOOL;        вҶҗ main contactor output
;      KM_STAR  : BOOL;        вҶҗ star contactor output
;      KM_DELTA : BOOL;        вҶҗ delta contactor output
;      FAULT    : BOOL;        вҶҗ fault status output
;      RUNNING  : BOOL;        вҶҗ motor running status
;  END_VAR
;
;  VAR
;      (* Instance attributes вҖ” internal variables *)
;      seal_in     : BOOL;     вҶҗ internal seal-in flag
;      star_timer  : TON;      вҶҗ timer instance inside FB
;      fault_latch : BOOL;     вҶҗ fault memory
;  END_VAR
;
;  (* Ladder logic inside FB вҖ” the CLASS METHODS *)
;  ...ladder rungs...
;
;  END_FUNCTION_BLOCK
;
; в”Җв”Җв”Җ I/O TAG TABLE в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
;
;  INPUTS  (X)
;  X0  вҶ’  START_PUMP        Pump start button
;  X1  вҶ’  STOP_PUMP         Pump stop button
;  X2  вҶ’  OL_PUMP           Pump overload
;  X3  вҶ’  START_CONV        Conveyor start button
;  X4  вҶ’  STOP_CONV         Conveyor stop button
;  X5  вҶ’  OL_CONV           Conveyor overload
;  X6  вҶ’  START_COMP        Compressor start button
;  X7  вҶ’  STOP_COMP         Compressor stop button
;  X10 вҶ’  OL_COMP           Compressor overload
;
;  OUTPUTS (Y)
;  Y0  вҶ’  KM_PUMP_MAIN      Pump main contactor
;  Y1  вҶ’  KM_PUMP_STAR      Pump star contactor
;  Y2  вҶ’  KM_PUMP_DELTA     Pump delta contactor
;  Y3  вҶ’  KM_CONV_MAIN      Conveyor main contactor
;  Y4  вҶ’  KM_COMP_MAIN      Compressor main contactor
;  Y5  вҶ’  KM_COMP_STAR      Compressor star contactor
;  Y6  вҶ’  KM_COMP_DELTA     Compressor delta contactor
;  Y7  вҶ’  FAULT_LAMP        Common fault lamp
;
;  FB INSTANCES (Objects)
;  PUMP_FB   : Motor_FB    вҶҗ pump = Motor()
;  CONV_FB   : Motor_FB    вҶҗ conv = Motor()
;  COMP_FB   : Motor_FB    вҶҗ comp = Motor()
;
;  TIMERS (inside FB вҖ” instance attributes)
;  T0  вҶ’  PUMP_STAR_TIMER   Pump star timer    (8s)
;  T1  вҶ’  COMP_STAR_TIMER   Compressor timer   (10s)
;
;  MEMORY (FB internal variables)
;  M0  вҶ’  PUMP_RUNNING      Pump status
;  M1  вҶ’  CONV_RUNNING      Conveyor status
;  M2  вҶ’  COMP_RUNNING      Compressor status
;  M10 вҶ’  PUMP_FAULT        Pump fault
;  M11 вҶ’  CONV_FAULT        Conveyor fault
;  M12 вҶ’  COMP_FAULT        Compressor fault
;
; в•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җ
;  LADDER DIAGRAMS вҖ” Main Program
;  (Each FB instance called like an object method)
; в•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җ

; в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
